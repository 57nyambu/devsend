## Template Features

### Placeholders Supported
Use these dynamic placeholders in your emails:
- `{{name}}` - Recipient's name
- `{{email}}` - Recipient's email
- `{{company}}` - Your company name
- `{{product}}` - Product name
- `{{link}}` - Call-to-action link
- `{{unsubscribe}}` - Unsubscribe URL
- `{{order_id}}` - Order number (transactional)
- `{{date}}` - Date field
- `{{total}}` - Total amount

## How to Use

### Creating a Professional Template

1. **Click "Create Template"**

2. **Select Template Type**
   - Choose from 6 template types
   - Or start with blank canvas

3. **Add Your Branding**
   - Logo URL (your hosted logo image)
   - Company name
   - Brand colors (optional customization)

4. **Configure Footer**
   - Company address
   - Contact phone
   - Website URL
   - Unsubscribe link

5. **Add Social Media**
   - Facebook page URL
   - Twitter profile URL
   - LinkedIn company page

6. **Click "Generate Professional Template"**
   - System creates fully-styled HTML
   - Includes all your branding
   - Ready to customize further

7. **Customize Content**
   - Edit the generated HTML
   - Add your specific messaging
   - Adjust placeholders as needed

8. **Save Template**
   - Give it a descriptive name
   - Add subject line with placeholders
   - Save for reuse

### Using Sample Templates

1. **Click "Load Sample Templates"**
   - Confirms you want to create samples
   - Creates 4 professional templates

2. **Browse Samples**
   - Look for "Sample" badge
   - Preview to see design

3. **Duplicate to Customize**
   - Click "Duplicate" on any sample
   - Modify for your needs
   - Save as your own template

### Sending with Templates

1. **Go to Send Page**
2. **Select Your Template**
3. **Choose Recipients**
4. **Placeholders Auto-Replace**
   - {{name}} becomes recipient name
   - {{email}} becomes recipient email
   - {{company}} becomes your company
5. **Click Send**

## Design Best Practices

### Logo Requirements
- **Format:** PNG or JPEG with transparent background
- **Size:** Recommended 200x50px to 400x100px
- **Hosting:** Must be publicly accessible URL
- **Alternative:** Use placeholder if no logo yet

### Email Dimensions
- **Width:** 600px maximum (email standard)
- **Height:** No limit, but keep reasonable
- **Images:** Always include alt text
- **File Size:** Keep total email under 100KB when possible

### Content Guidelines
- **Subject Line:** 40-50 characters ideal
- **Preview Text:** First line shows in inbox
- **Headline:** Clear, benefit-focused
- **Body Copy:** Short paragraphs, scannable
- **CTA:** One primary call-to-action
- **Footer:** Always include unsubscribe

### Mobile Optimization
All templates are mobile-responsive:
- Text resizes for small screens
- Buttons are touch-friendly (44px minimum)
- Images scale proportionally
- Padding adjusts for mobile

### Accessibility
Templates include:
- Semantic HTML structure
- Alt text placeholders for images
- Sufficient color contrast
- Readable font sizes
- Clear link styling

## Troubleshooting

### Logo Not Showing
- ✓ Verify URL is publicly accessible
- ✓ Check image format (PNG/JPEG)
- ✓ Try HTTPS URL instead of HTTP
- ✓ Use image hosting service (Imgur, Cloudinary)

### Styles Not Applying
- ✓ Email clients strip some CSS
- ✓ Use inline styles when possible
- ✓ Avoid external stylesheets
- ✓ Test in multiple email clients

### Placeholders Not Replacing
- ✓ Use exact syntax: {{name}} not {name}
- ✓ Check spelling matches placeholder list
- ✓ Ensure recipient data has that field
- ✓ Review email logs for errors

### Mobile Layout Issues
- ✓ Templates are responsive by default
- ✓ Avoid fixed widths in custom code
- ✓ Test on actual mobile device
- ✓ Check media queries in styles
